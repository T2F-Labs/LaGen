% Custom hybrid cover that uses template colors
\newcommand{\hybridCover}[5]{%
  % #1: Main Title
  % #2: Subtitle  
  % #3: Author
  % #4: Organization
  % #5: Date
  \thispagestyle{empty}
  \begin{tikzpicture}[remember picture,overlay]
    % Use template 2's corporate colors but custom layout
    \fill[brandPrimary!5] (current page.north west) rectangle (current page.south east);
    
    % Custom geometric header using template colors
    \fill[brandPrimary] (current page.north west) rectangle ([yshift=-3cm]current page.north east);
    \fill[brandSecondary!80] ([yshift=-3cm]current page.north west) rectangle ([yshift=-3.5cm]current page.north east);
    
    % Custom title area with template color scheme
    \fill[white, opacity=0.95] ([xshift=4cm, yshift=-8cm]current page.north west) 
      rectangle ++(12cm,5cm);
    
    % Brand-consistent title styling
    \node[color=brandPrimary, font=\Huge\bfseries, text width=11cm, align=center] 
      at ([xshift=10cm, yshift=-6.5cm]current page.north west) {#1};
      
    \node[color=brandSecondary, font=\LARGE, text width=11cm, align=center] 
      at ([xshift=10cm, yshift=-8cm]current page.north west) {#2};
    
    % Custom decorative elements using template colors
    \foreach \i in {0,45,90,135,180,225,270,315}
      \fill[rotate around={\i:(2,2)}, brandAccent, opacity=0.3] 
        ([xshift=2cm, yshift=-2cm]current page.north west) ellipse (0.5cm and 0.2cm);
    
    % Author info with hybrid styling
    \node[color=primary, font=\large, align=center] 
      at ([xshift=10cm, yshift=-15cm]current page.north west) 
      {\textbf{Author:} \textcolor{brandSecondary}{#3}\\[0.3cm]
       \textbf{Organization:} \textcolor{brandSecondary}{#4}\\[0.3cm]
       \textbf{Date:} \textcolor{brandSecondary}{#5}};
  \end{tikzpicture}
  \newpage
}

\section{Design Strategy}

The hybrid approach allows maximum flexibility by combining:
\begin{enhancedlist}
    \item Custom cover page design for unique branding
    \item Template body styling for consistent professional appearance
    \item Shared color palette for visual coherence
    \item Modular feature loading for optimal performance
\end{enhancedlist}

\section{Color Harmony}

The hybrid approach maintains color harmony by:
\begin{itemize}
    \item Using template color definitions
    \item Applying custom layouts with consistent colors
    \item Leveraging brand color cascade system
    \item Ensuring visual continuity between cover and content
\end{itemize}
