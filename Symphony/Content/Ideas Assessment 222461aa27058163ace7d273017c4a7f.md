## Workload and Responsibilities

### Developer Productivity Tools

- **Percentage of Workload**: 35% Frontend, 30% Backend, 35% AI
- **Team Division**: 2 Frontend, 1 Backend, 2 AI
- **Responsibilities**:
    - **Backend**:
        - Design and maintain APIs for ERD
        - Save/load/export ERD
        - SQL code generation engine
        - Session/workspace management
        - Integration with DB systems
        - Custom file extension building
    - **Frontend**:
        - Build dynamic canvas for ERD drawing
        - Design elegant, responsive UI for diagramming tools
        - Shape libraries (ERD, use case, class diagrams)
        - Keyboard/drag-drop shortcuts
        - Desktop distribution via Electron or Tauri
    - **AI**:
        - LLM-based prompt-to-diagram generation
        - Computer Vision: extract diagrams from images or PDFs
        - Fine-tune/curate ERD dataset for training
        - Autocomplete/validate diagrams using AI heuristics
        - Intent detection & diagram suggestion

### Agentic IDE

- **Percentage of Workload**: 20% Frontend, 30% Backend, 50% AI
- **Team Division**: 2 Backend, 3 AI

### Text Dataset Generator

- **Percentage of Workload**: 10% Frontend, 20% Backend, 55% AI, 15% Research
- **Team Division**: 1 Frontend, 1 Backend, 1 AI, 1 Research
